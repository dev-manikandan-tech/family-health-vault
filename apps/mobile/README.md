# apps/mobile

React Native Expo application for patients, family members, and caregivers.

## Responsibilities

- Authentication flows (Google, Apple, Email OTP)
- Family and person management
- Medical visit and document timeline
- Offline-first document viewer and uploader
- Push notifications and background sync
- Tablet and accessibility (AA) support

## Tech

- React Native Expo
- Expo Router
- WatermelonDB + SQLite
- React Query / Zustand
- Tamagui or React Native Paper (TBD)
